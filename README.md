# Snake Game

A classic Snake game built with vanilla HTML, CSS, and JavaScript using the Canvas API.

## How to Play

1. Open `snake/index.html` in your browser
2. Use **Arrow keys** or **WASD** to move the snake
3. Eat the yellow food to grow and earn points
4. Avoid hitting the walls or yourself
5. Press **Space** to restart after game over

## Features

- Smooth canvas-based rendering
- Score tracking
- Progressive speed increase as you score
- Clean, dark-themed UI
